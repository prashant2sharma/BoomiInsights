Note : Even if Boomi Insights is executed successfully, it is still necessary to run the Naming Conventions. Both sections must display green signal to confirm successful completion.
With Boomi Insights, you can ensure that your processes are compliant with the necessary standards. By using the Boomi Insights parameter, you can analyze your processes to see if they meet compliance requirements. You have the flexibility to input data either through a CSV file or a JSON payload. Once you provide the data, Boomi Insights generates a detailed report that shows the accuracy of your processes. This report helps you identify areas where your processes are compliant and highlights any discrepancies, allowing you to make the necessary adjustments. With Boomi Insights, you can maintain high levels of accuracy and compliance in your operations.
The Naming Conventions tool is your go-to for maintaining consistency and clarity in your processes. It checks if all the components in your process follow the prescribed naming conventions. You can input data using either a CSV file or a JSON payload, which makes it very convenient. The tool then provides a comprehensive report that covers both components and shapes, along with their locations. This report helps you spot any deviations from the naming conventions, enabling you to rectify them promptly. By ensuring that all elements follow the proper naming conventions, this tool aids you in maintaining organized and easily understandable processes, which is crucial for effective communication and management within your organization.
